# Rubric: Discounts (Weight: 8)

Promotional and volume discounts off net pricing. Distinct from rebates (front-end vs back-end). Higher discount = lower landed cost = better margin on each transaction.

## 0–10 Scale

| Score | Threshold |
|-------|-----------|
| 10 | ≥20% discount |
| 9 | 15–19% |
| 8 | 12–14% |
| 7 | 10–11% |
| 6 | 8–9% |
| 5 | 6–7% |
| 4 | 4–5% |
| 3 | 2–3% |
| 2 | <2% |
| 1 | Discounts exist but rarely honored in practice |
| 0 | No discount program |

## Notes

- These are promotional / order discounts (show specials, stocking orders, promo periods), not standard trade discounts already baked into net price.
- "Net price" already reflects standard dealer discount; this rubric scores the *additional* promotional layer.
- Seasonal promos, new-product launch discounts, show specials all count.
- If discounts require large MOQs that Accent can't hit, score one tier lower.
