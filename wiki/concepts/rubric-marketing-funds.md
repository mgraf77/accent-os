# Rubric: Marketing Funds (Weight: 7)

Co-op / MDF (Market Development Funds) plus digital asset quality. Composite score: MDF%, asset quality, portal access, co-branded materials.

## Composite scoring (RUBRIC_COMPONENTS)

Points breakdown (max 11, capped at 10):

| Component | Points |
|-----------|--------|
| MDF ≥3% | 5 pts |
| MDF 2–2.9% | 4 pts |
| MDF 1.5–1.9% | 3 pts |
| MDF 1–1.4% | 2 pts |
| MDF 0.5–0.9% | 1 pt |
| Asset quality: pro (full kit + video) | 4 pts |
| Asset quality: rich (images + copy) | 3 pts |
| Asset quality: standard (logos + images) | 2 pts |
| Asset quality: minimal (logos only) | 1 pt |
| Self-serve portal | 1 pt |
| Co-branded materials | 1 pt |

## 0–10 Rubric guide

| Score | Combination |
|-------|-------------|
| 10 | MDF 5pts + rich assets + portal + cobrand |
| 9 | MDF 5pts + rich assets + 1 bonus |
| 8 | MDF 4pts + rich assets |
| 7 | MDF 3pts + standard/rich assets |
| 6 | MDF 2pts + standard assets + portal |
| 5 | MDF 1pt + standard assets (average) |
| 4 | MDF 1pt + minimal assets |
| 3 | Token MDF + 1 asset type |
| 2 | Assets only, no MDF |
| 1 | Minimal asset library |
| 0 | No MDF, no assets |

## Notes

- MDF % is of annual purchases. 2% on $500K spend = $10K/year in marketing support.
- High-quality digital assets (hi-res images, cut sheets, videos) reduce Accent's content production cost.
- Portal access (vendor's own dealer portal) means self-serve asset requests without emailing the rep.
- Co-op claims often require prior approval + invoice submission — track in AccentOS Co-op Tracker.
