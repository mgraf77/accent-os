# Rubric: Returns / RGA (Weight: 7)

Return Goods Authorization policy. Covers restock fees, return windows, and process friction. Critical for managing floor model turnover, customer returns, and warranty claims.

## 0–10 Scale

| Score | Policy |
|-------|--------|
| 10 | Free returns, no restock fee, process <7 days |
| 9 | No restock fee, window <14 days |
| 8 | ≤10% restock fee, quick process |
| 7 | ≤15% restock fee, 2–3 week turnaround |
| 6 | ≤20% restock fee, slow |
| 5 | ≤25% restock fee (average) |
| 4 | 25–30% restock fee |
| 3 | >30% restock fee |
| 2 | Case-by-case approval only |
| 1 | Rare approvals, effectively no returns |
| 0 | No returns accepted |

## Notes

- Restock fees compound with freight cost — a 20% restock + paying return freight can approach 30% total loss.
- Some vendors require defective items be photographed/inspected before issuing RGA — adds weeks.
- Lighting One membership sometimes unlocks better return terms.
- Display buybacks (see [[rubric-display]]) are separate from standard returns.
