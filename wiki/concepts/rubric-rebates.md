# Rubric: Rebates (Weight: 10)

Highest-weight financial category. Rebates are year-end backend payments from vendor to dealer based on total purchase volume. Directly improves net margin without affecting list pricing.

## 0–10 Scale

| Score | Threshold |
|-------|-----------|
| 10 | ≥6% rebate |
| 9 | 5–5.9% |
| 8 | 4–4.9% |
| 7 | 3–3.9% |
| 6 | 2.5–2.9% |
| 5 | 2–2.4% |
| 4 | 1.5–1.9% |
| 3 | 1–1.4% |
| 2 | <1% |
| 1 | Exists but effectively useless (too complex/unattainable) |
| 0 | No rebate program |

## Notes

- Industry average for Accent's vendor tier: ~2–3% (score 5–7).
- Tiered rebates common: first $X purchases at 1%, next tranche at 2%, etc. Score the blended expected rate.
- Some vendors offer rebates only to Lighting One members — factor in if Accent is/isn't qualified.
- Rebates payable January–March for prior calendar year. Cash flow timing matters for score adjustment.
