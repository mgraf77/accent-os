# Rubric · Rebates

**Weight:** MEDIUM. **What it measures:** annual purchase-volume rebate Accent earns back from the vendor.

| Score | Meaning |
|---|---|
| **10** | ≥5% of annual purchases, automatic accrual |
| **8** | 3-5% |
| **6** | 1-3% |
| **4** | <1% OR threshold-gated and rarely hit |
| **2** | Token amount |
| **0** | No rebate program |

Tracked in `coop_tracker` table (Track 2.3 / M02). See [[sop-coop-claim]].
