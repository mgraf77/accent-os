# Rubric · Returns Policy

**Weight:** MEDIUM. **What it measures:** vendor's flexibility on returns of unsold inventory + returns of defective product.

| Score | Meaning |
|---|---|
| **10** | Accepts returns, no restock fee, no return window cap on defective |
| **8** | Accepts returns, ≤10% restock fee, 90+ day window |
| **6** | Accepts returns, 15-25% restock, 30-90 day window |
| **4** | Accepts returns conditionally (defective only, OR with prior approval) |
| **2** | Returns extremely limited |
| **0** | No returns accepted |
