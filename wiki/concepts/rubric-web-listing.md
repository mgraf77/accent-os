# Rubric · Web Listing

**Weight:** MEDIUM. **What it measures:** Accent's positioning on the vendor's official "Find a Dealer" / dealer locator.

| Score | Meaning |
|---|---|
| **10** | Featured / Premier dealer on vendor site, top of regional results |
| **8** | Listed prominently, no Premier badge |
| **6** | Listed but not differentiated |
| **4** | Listed but with stale info OR low ranking |
| **2** | Listed but pointing at the wrong location / phone |
| **0** | Not listed OR vendor has no locator |

## State (2026-05-01 rescore)

Top-30 vendors by sales scored via direct web research. 226 lower-volume vendors bulk-assigned `score=3` (deferred). See [[sources/session-log]] entry "2026-05-01 web listing rescore".
