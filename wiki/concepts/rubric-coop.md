# Rubric · Co-op

**Weight:** MEDIUM. **What it measures:** co-op funds offered as % of purchases, tied to specific marketing/display use.

| Score | Meaning |
|---|---|
| **10** | ≥2% accrual, easy claims, flexible spend |
| **8** | 1-2%, claims approved on receipt |
| **6** | 0.5-1%, claims with paperwork |
| **4** | <0.5% OR specific-use-only |
| **2** | Token amount |
| **0** | No co-op |

Tracked alongside rebates in `coop_tracker` table. Daily Brief surfaces deadlines ≤30d via the Co-op Deadlines tile (Track 1.3 phase 2). See [[sop-coop-claim]].
