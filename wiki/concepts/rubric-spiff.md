# Rubric · Spiff

**Weight:** MEDIUM. **What it measures:** sales-rep-direct incentive on a vendor's products.

| Score | Meaning |
|---|---|
| **10** | >15% spiff, all products, simple claim |
| **8** | 10-15% spiff, most products |
| **6** | 5-10% spiff, qualifying products |
| **4** | <5% spiff OR limited categories |
| **2** | Token amount |
| **0** | No spiff program |

## Reference programs (knowledge engine)

- Kuzco — 3%
- LEDI — 2%
- Hinkley Honors

> Spiff is separate from [[rubric-discount]]. Don't double-count.
