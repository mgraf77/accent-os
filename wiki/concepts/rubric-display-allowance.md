# Rubric · Display Allowance

**Weight:** MEDIUM. **What it measures:** dedicated dollars for in-showroom product displays.

| Score | Meaning |
|---|---|
| **10** | Dedicated $$, paid on schedule, refreshes on cycle |
| **8** | Dedicated $$, paid eventually, manual refresh request |
| **6** | Display program exists, paid against specific build-out |
| **4** | Program exists but rare actual disbursement |
| **2** | Program named but Accent has never collected |
| **0** | No allowance |

Tracked in `showroom_displays` table (Track 5.8, M25). Active displays + co-op linkage live on each `wiki/entities/vendors/<slug>.md` page.
