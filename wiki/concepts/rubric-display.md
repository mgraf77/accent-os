# Rubric: Display Program (Weight: 6)

Showroom display support: discount on display product, promotional cadence, and buyback/refresh program. A vendor with a 50%+ display discount earns a score of 8; 60%+ display discount with monthly promos earns a score of 10.

## Composite scoring (RUBRIC_COMPONENTS)

Points breakdown (max 10):

| Component | Points |
|-----------|--------|
| Display discount ≥60% | 6 pts |
| Display discount ≥50% | 5 pts |
| Display discount ≥35% | 3 pts |
| Display discount ≥25% | 2 pts |
| Display discount ≥10% | 1 pt |
| Promo cadence: monthly+ | 2 pts |
| Promo cadence: quarterly | 2 pts |
| Promo cadence: occasional | 1 pt |
| Buyback: generous (funded refresh) | 2 pts |
| Buyback: standard (annual swap) | 2 pts |
| Buyback: limited (case by case) | 1 pt |

## 0–10 Rubric guide

| Score | Combination |
|-------|-------------|
| 10 | 60%+ disc + monthly promo + generous buyback |
| 9 | 60%+ disc + quarterly promo + standard buyback |
| 8 | 50%+ disc + monthly promo + standard buyback |
| 7 | 35%+ disc + quarterly promo + standard buyback |
| 6 | 25–35% disc + occasional promo + standard buyback |
| 5 | 25% disc + occasional promo + limited buyback (average) |
| 4 | 25% disc + no promo + limited buyback |
| 3 | 10–25% disc + occasional promo + no buyback |
| 2 | Any disc + no promo + no buyback |
| 1 | <25% disc + no promo + no buyback |
| 0 | No display program |

## Notes

- Buyback/refresh programs are critical for floor model turnover. Stale displays hurt sales.
- "Generous" buyback = vendor funds replacement fixtures or offers 1-for-1 swap annually.
- Promo cadence drives foot traffic + promotional email opportunities.
- High display discount (50%+) makes it economical to display lower-velocity product.
