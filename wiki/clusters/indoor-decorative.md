# Indoor Decorative Lighting Cluster

## Overview

Core residential and hospitality product family at Accent Lighting. Covers all interior decorative fixtures sold through the showroom and e-commerce channels. Highest quote frequency and showroom floor footprint.

## Product categories

| category | typical SKUs | price range | key vendors |
|---|---|---|---|
| Pendants | 200–400 | $80–$2,500 | varies by vendor family |
| Chandeliers | 80–150 | $200–$8,000+ | high-tier brands |
| Flush / Semi-flush | 150–300 | $60–$800 | mid-tier volume |
| Wall sconces | 200–350 | $50–$1,200 | residential + staging |
| Vanity strips | 100–180 | $80–$600 | bath/spec |
| Table lamps | 120–200 | $60–$400 | staging + gifting |
| Floor lamps | 60–100 | $120–$600 | staging + showroom |

## Selling notes

- **Layered lighting design** is the core upsell frame: ambient (ceiling), task (vanity/under-cabinet), accent (sconce), decorative (statement piece). Quote all four layers, not just the request item.
- **Finish coordination**: customers buying chandeliers almost always need matching sconces and vanity. Prompt in the quote to reduce return visits.
- **Bulb / driver ecosystem**: some vendor families require proprietary LED drivers. Flag in quote notes to avoid field warranty issues.
- **Showroom display cross-sells**: reference active showroom displays (Showroom Displays module) when a customer wants to see a fixture in person.

## Quoting tips

- Lead with the ceiling anchor (pendant / chandelier) — then fill in the rest of the room.
- Use Price Book to check current margin before discounting; decorative fixtures often have wider margin than commodity.
- For designers and trade partners: reference Trade Partners module for designer discount structures.
- Warranty risk: cosmetic claims (finish chips, shade color mismatch) are most common in this category — set severity = cosmetic in the Warranty module.

## Inventory notes

- Reorder lead time: 4–16 weeks depending on vendor and origin (domestic vs import).
- Use Demand Forecast module to project reorder quantities based on PO history.
- Low-stock threshold: decorative items with qty_on_hand = 0 that are active showroom displays should trigger immediate reorder.
