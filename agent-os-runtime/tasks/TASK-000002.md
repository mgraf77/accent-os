# Build Event Logger

Implement append-only JSONL event logging for the AgentOS Runtime.

## Objective

Create `runtime/event_log.py` — the immutable audit trail for all task lifecycle
events. This module is critical for replay, debugging, and observability.

## Deliverables

- Append-only writes to `events/YYYY-MM-DD.events.jsonl`
- Event ID format: `EVT-{6-digit-zero-padded-counter}`
- Full set of event type constants
- `log_event()`, `read_events()`, `read_all_events()` functions

## Dependencies

Requires TASK-000001 (timestamp engine) for UTC-aware datetime generation.

## Notes

**Never overwrite or truncate the events log.** All writes are append-only.
The event counter is derived by counting existing lines in today's log file
before each write.
