# Build Timestamp Logger

Implement the core timestamp evaluation engine for AgentOS Runtime.

## Objective

Create `runtime/timestamp_engine.py` with UTC-aware datetime functions that power
the task scheduling system. This is a foundational module with no dependencies on
other AgentOS components.

## Deliverables

- `get_now_utc()` — always returns current UTC datetime with tzinfo
- `is_trigger_ready(task, now)` — returns True if trigger_at is unset or in the past
- `get_trigger_delta(task, now)` — returns timedelta to trigger (negative = past)

## Notes

All datetimes must be timezone-aware UTC objects. The system uses ISO 8601 strings
in YAML frontmatter, so string parsing with normalisation of the trailing `Z` suffix
is required.
